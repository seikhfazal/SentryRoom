# Design Notes

Sentinel Room uses an original dark command-center style inspired by common desktop app patterns: persistent sidebar on desktop, bottom navigation on mobile, glass panels, scan-line motion, terminal event feeds, and strong red/yellow/green/blue state colors.

The product language avoids weapon framing. Demo mode is theatrical, but every control is bounded by safety rules and timeout behavior.

Design goals:

- Look like a proper installed control app.
- Work from a Windows browser, desktop app, Android browser, and iPhone browser.
- Keep phone controls large and simple.
- Make Stop All obvious everywhere.
- Keep advanced risky outputs hidden behind confirmations.
