# Desktop App

The desktop app is built with Tauri and the shared React UI.

Features:

- Windows app target.
- Splash/loading placeholder.
- Backend connection setting.
- Offline/disconnected state.
- Open Web Controller button in Settings.
- Show QR Code placeholder in Settings.

Build:

```powershell
cd E:\SentinelRoom\apps\desktop
npm install
npm run tauri:build
```
