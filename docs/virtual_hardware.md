# Virtual Hardware

Use Wokwi first:

```text
E:\SentinelRoom\simulation\wokwi
```

The simulation includes ESP32, servos, pseudo-laser LED, RGB LEDs, buzzer, PIR button, door button, ultrasonic sensor, and serial output.

Limitations:

- MQTT is mocked with serial output in the starter simulation.
- Exact servo power behavior is not simulated.
- RGB rings may be simplified to RGB LEDs.

Use Tinkercad Circuits only as a fallback for a simpler Arduino Uno style demonstration.
